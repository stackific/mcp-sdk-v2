# Hard rules
- **Nothing in this project may attribute authorship to Claude or any AI** —
  not commit messages, not commit trailers, not PR titles or descriptions, not
  issues, not code comments, not docs. Never write "committed by Claude", never
  add a `Co-Authored-By: <AI>` trailer, and never add a "Generated with <AI>"
  footer or badge of any kind. This rule is not scoped to commits; it covers
  every artifact the project produces.
- All code must have automated tests and you must not cut-corners, omit/skip to bypass the actual fixes needed in the code
- Use 2 spaces as indentation
- Must write modern idiomatic Node/TypeScript code
- Must use `pnpm` as the package manager
- Inline documentation is a must. Try to use 1st-party official or well-maintained packages as much as possible.
